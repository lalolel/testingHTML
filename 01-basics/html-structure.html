HTML Document Structure

Lesson 1: Understanding the anatomy of an HTML document

1. The DOCTYPE Declaration

Every HTML document must start with a DOCTYPE declaration:

<!DOCTYPE html>
Why it's important: The DOCTYPE tells the browser which version of HTML you're using and ensures the page renders correctly.

2. The HTML Element

The <html> element is the root element that contains all other elements:

<html lang="en">
  • The lang attribute specifies the language of the document
  • This helps screen readers and search engines
  • Common values: "en" (English), "es" (Spanish), "fr" (French)

3. The Head Section

The <head> contains metadata about the document:

  • Meta charset: Defines character encoding (UTF-8 is standard)
  • Viewport: Controls how the page is displayed on mobile devices
  • Title: Appears in browser tab and search results
  • Description: Used by search engines
  • CSS and JavaScript: Links to external files or embedded code

4. The Body Section

The <body> contains all visible content:

  • Text content
  • Images and media
  • Interactive elements
  • Navigation and structure
Remember: Everything users see and interact with goes in the body!

5. Complete Structure Overview

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Visible content goes here -->
</body>
</html>
