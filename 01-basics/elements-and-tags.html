HTML Elements and Tags

Lesson 2: Understanding the building blocks of HTML

What are HTML Elements?

HTML elements are the building blocks of web pages. They consist of tags that tell the browser how to structure and display content.

Anatomy of an HTML Element

<p>This is a paragraph</p>
  • Opening tag: <p>
  • Content: "This is a paragraph"
  • Closing tag: </p>

Types of HTML Elements

1. Container Elements (Paired Tags)

These elements have both opening and closing tags and can contain content or other elements.

<h1>Main Heading</h1>
<p>This is a paragraph with <strong>bold text</strong> inside.</p>
<div>This is a container</div>
Result:

Main Heading

This is a paragraph with bold text inside.

This is a container

2. Self-Closing Elements (Void Elements)

These elements don't have closing tags and cannot contain other elements.

<br> <!-- Line break -->
<hr> <!-- Horizontal rule -->
<img src="image.jpg" alt="Description">
<input type="text" name="username">
Result:
Line 1
Line 2 (after line break)

Nesting Elements

HTML elements can be nested inside other elements. Proper nesting is crucial for valid HTML.

✅ Correct Nesting

<div>
  <h2>Section Title</h2>
  <p>This paragraph contains <strong>bold text</strong> and <em>italic text</em>.</p>
</div>

❌ Incorrect Nesting

<p>This is <strong>bold and <em>italic</strong> text</em>.</p>
Remember: Tags must be closed in the reverse order they were opened. Think of it like parentheses in math!

Block vs Inline Elements

Block Elements

Take up the full width available and start on a new line.

<div>Block element 1</div>
<div>Block element 2</div>
<p>Another block element</p>
Block element 1
Block element 2

Another block element

Inline Elements

Only take up as much width as necessary and don't start on a new line.

<span>Inline 1</span> <span>Inline 2</span> <strong>Bold</strong> <em>Italic</em>
Inline 1 Inline 2 Bold Italic

HTML Comments

Comments allow you to add notes in your HTML that won't be displayed to users.

<!-- This is a comment -->
<p>This paragraph is visible</p>
<!--
  This is a multi-line comment
  that spans several lines
-->
Use comments for:
  • Explaining complex code
  • Temporarily disabling code
  • Adding TODO notes
  • Marking sections of your document
